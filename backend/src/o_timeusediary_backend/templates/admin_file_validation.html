File Validation

Validate studies and activities files in-memory without creating or deleting studies.

Last Updated: {{ current_time.strftime('%Y-%m-%d %H:%M:%S UTC') }}

1) Single activities file

Checks one activities JSON file only (schema + internal validations).

Tip: language-cross checks are skipped in this mode.

2) Activities cross-language set

Checks consistency across one activities file per language.

3) Full study package

Checks one selected study from studies_config plus all related language activities files.

If multiple studies are found, select one and validate again.

By default, filenames from studies_config are used to map uploaded files.
