URL encoder for return_url

Paste the raw destination URL below. The tool encodes it as a single query-parameter value so it can be safely appended to invitation links as return_url=....

Encoding rule

Use standard percent-encoding for one query-parameter value. Recommended methods:

  • JavaScript: encodeURIComponent(rawReturnUrl)
  • Python: urllib.parse.quote(raw_return_url, safe='')

Example invitation link:

index.html?study_name=default&pid=PARTICIPANT_ID&return_url=https%3A%2F%2Fexample.org%2Ffinish%3Ftoken%3Dabc123

Do not double-encode the value. Encode only the raw destination URL once.
